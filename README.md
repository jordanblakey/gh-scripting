## Installation

```sh
# note: Github actions use dash shell, which is a lightweight POSIX compliant bourne shell alternative.
# note: Use #!/usr/bin/env sh

# install gh command
./install.sh

# CORE
gh --help
# auth | browse | codespace | gist | issue | org | pr | co | project | release | repo

# HELP TOPICS
```
